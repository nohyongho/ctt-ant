
# AIRCTT ONE-PAGE MASTER BLUEPRINT

## 1. AIRCTT 전체 구조 (7대 핵심 모듈)
1) 장터(마켓플레이스)
2) 쿠폰 시스템(지갑/업그레이드/반경)
3) 이벤트 시스템(5단계 쿠폰잡기)
4) 테이블오더(식당/로드매장)
5) 문화예술(공연/영화/연극/전시)
6) 머천트(사업자 페이지)
7) 고객DB(자동등록)

## 2. 전체 데이터 흐름
[사용자] -> 쿠폰 잡기(5단계) -> [쿠폰지갑 자동 업데이트] & [해당 가맹점 고객DB 자동 등록]
[장터] -> 카테고리 선택 -> 매장 상세 -> 쿠폰 자동 적용 -> 결제 -> 주문/티켓 발급

## 3. 머천트(사업자) 핵심 6기능
1. 매장정보 관리
2. 메뉴・상품・공연 등록
3. 홈페이지 URL 입력
4. 쿠폰 발행(파일 업로드 포함)
5. 공지/이벤트 등록
6. 테이블QR / 좌석맵 / 회차관리

## 4. AIRCTT만의 차별화 기능 (6개)
1. 쿠폰 잡기 → 고객DB 자동등록
2. 쿠폰 업그레이드(지갑 자동 정리)
3. 반경 km 설정(사업자 쿠폰 발행)
4. 테이블 QR 자동 생성 기능
5. 문화예술 전용 입력값(공연/좌석/회차/포스터)
6. 홍보 영상/포스터 파일 업로드 강화

## 6. 엔티 개발 우선순위 (Step 1 ~ 10)

### 🔵 Step 1 — 기본 프레임 구축 (Current)
- 장터 메인
- 카테고리 구조
- 매장 리스트
- 매장 상세 페이지(빈 틀)

### 🔵 Step 2 — 머천트 기본 구조
- 가맹점 등록 페이지
- 메뉴/상품 입력
- 매장 정보(영업시간/설명/대표이미지)

### 🔵 Step 3 — 쿠폰 시스템
- 쿠폰 발행 UI
- 쿠폰 파일 업로드
- 지갑 구조
- 업그레이드/삭제 로직
- 쿠폰 자동등록 RULE

### 🔵 Step 4 — 이벤트 시스템(5단계)
- 이벤트 페이지
- 단계별 애니메이션
- 반경 km 노출
- 쿠폰 발행 연동

### 🔵 Step 5 — 테이블오더
- 테이블 번호 생성
- QR 자동 생성
- 메뉴 → 장바구니 → 결제
- 주방 모니터
- 첫 방문 할인 설정

### 🔵 Step 6 — 결제(PG/VAN)
- 연동 자리 미리 잡고 무료 모드 구축

### 🔵 Step 7 — 문화예술 시스템
- 공연/영화 등록 페이지
- 회차/날짜, 좌석맵, 가격 테이블
- 포스터/티저 업로드

### 🔵 Step 8 — 문화예술 티켓 구매
- 좌석 선택, 쿠폰 적용, 결제, 티켓(QR) 생성

### 🔵 Step 9 — 고객 DB 자동화
- 쿠폰잡기/오더/티켓 → 자동 저장 로직 통합

### 🔵 Step 10 — 알림/푸시/이벤트 자동화
- 가맹점이 고객에게 쿠폰 발송
